# Asset Dashboard

個人資產配置追蹤工具，支援五桶框架（核心／攻擊／分散／另類／防禦），自動抓取 Yahoo Finance 報價與台銀匯率，提供時間加權報酬（TWR）分析、退休規劃試算與再平衡助手。

## 功能概覽

### 7 個分析頁籤
| 頁籤 | 內容 |
|------|------|
| 資產分布 | 五桶比例圓餅圖、總資產、持倉明細、五桶偏離警示 |
| 資產走勢 | 歷史折線圖（總資產 / 五桶比例 / 個股走勢，三模式切換） |
| 績效分析 | TWR 年化/累計/YTD/近一年、NAV 成長曲線、年度/月度報酬長條圖 |
| 退休規劃 | 軌跡圖（實際/預測/所需）、預計達成年份、里程碑表（年度/月度切換） |
| 再平衡分析 | 缺口分析（全部/台幣帳戶/美金帳戶視角）＋ 新資金分配助手 |
| 持倉明細 | 五桶視角 ↔ 帳戶視角切換，價格與目標比例可直接行內編輯 |
| 交易紀錄 | 所有交易依日期倒序、支援年月篩選、可編輯/刪除 |

### 核心特性
- **自動更新報價**：開啟頁面自動從 Yahoo Finance 抓股價，同步台銀即期匯率（USD/TWD）
- **每日快照**：自動記錄當日總資產、五桶比例、各標的市值與股數
- **補單回補歷史**：新增或編輯過去日期的交易，系統自動修補所有歷史快照，確保 TWR 計算準確
- **TWR 時間加權報酬**：排除現金流入/流出影響，正確衡量投資績效（非簡單報酬率）
- **新資金分配助手**：輸入可投入金額，系統依各桶缺口自動計算各標的買入股數（台股以千股為單位，美股以整股）
- **模糊模式**：一鍵遮蔽所有金額數字，方便截圖分享
- **匯入/匯出 JSON**：完整備份與還原整份投資組合

### 交易類型
`buy`（買入）、`sell`（賣出）、`cash_in`（現金存入）、`cash_out`（現金提出）、`new_position`（建立新標的）、`new_cash_account`（建立現金帳戶）、`transfer`（帳戶轉帳，支援同幣/跨幣）

## 系統需求

- [Node.js](https://nodejs.org/) v18 以上
- npm（隨 Node.js 附帶）

## 安裝與啟動

```bash
# 1. clone 專案
git clone https://github.com/Shiauka/asset-dashboard.git
cd asset-dashboard

# 2. 安裝套件
npm install

# 3. 啟動開發伺服器
npm run dev
```

開啟瀏覽器前往 [http://localhost:3000](http://localhost:3000)

## 首次設定

1. 點選右上角「⚙ 根目錄設定」
2. 輸入一個本機資料夾路徑（例如 `C:\MyData\AssetSnapshots`），用來存放每日快照 JSON
3. 設定完成後，每次開啟頁面會自動載入最新快照並更新報價

> 若不設定根目錄，資料只存在 localStorage（關閉瀏覽器後仍保留，但換裝置會遺失）。

## 資料儲存

| 儲存位置 | 內容 | 說明 |
|----------|------|------|
| 瀏覽器 localStorage | 目前狀態 | 即時，無需設定 |
| 本機資料夾（自訂路徑） | 每日 JSON 快照 | 格式 `YYYY-MM-DD.json`，由根目錄設定決定路徑 |

**所有資料存在本機，不會上傳任何伺服器。**

## 五桶框架

| 桶別 | 預設目標比例 | 說明 |
|------|-------------|------|
| 核心 | 35% | 主要寬基 ETF（如 VOO、0050） |
| 攻擊 | 30% | 成長型 ETF（如 QQQ、00631L） |
| 分散 | 15% | 已開發/新興市場（如 VEA、VWO） |
| 另類 | 5% | 另類資產（如 IBIT、IAU） |
| 防禦 | 15% | 短債與現金（如 SGOV、台幣活存） |

## 注意事項

- 本工具設計為**本機自架**，不支援部署至 GitHub Pages（需要 Node.js 伺服器執行 API routes）
- 每次啟動 `npm run dev` 後才能使用，關閉終端機則停止服務
- 若要在同區網路其他裝置（如手機）存取，可改用：
  ```bash
  npm run dev -- --hostname 0.0.0.0
  ```
  再從手機連 PC 的區網 IP
